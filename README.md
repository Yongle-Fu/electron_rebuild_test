# electron-cmsn

## Run the app

```javascript
cd electron-cmsn
npm i
npm i pc-ble-driver-js --runtime=electron --build-from-source
npm i --runtime=electron --target=76 serialport
DEBUG=* npm run start
// npm i --runtime=electron --target=75 serialport
// npm i --runtime=electron --target=89 serialport
// npm i --runtime=electron serialport --build-from-source
```
